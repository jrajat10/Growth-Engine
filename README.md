# Growth Intelligence Engine

A closed-loop paid acquisition and growth decisioning platform — built for AMER growth teams running weekly optimization cycles.

## What it does

- **Weekly Brief** — narrative-first summary of what happened, what matters, and what to do next
- **Budget Allocation** — AI-powered channel scoring within finance guardrails; every recommendation requires approval
- **Funnel Analytics** — full visibility from visit to paying user with CAC, quality-CAC, and activation rates
- **Pacing & Targets** — quarterly revenue targets decomposed into per-channel weekly signup goals
- **Experimentation** — incrementality-first framework: geo holdouts and A/B tests gating budget decisions
- **Creative Insights** — performance signals surfaced to your creative team; no copy generated by the engine
- **Competitive Intelligence** — competitor signals tied to observed metric changes, with confidence scoring
- **Decision Ledger** — full audit trail of every recommendation made and approved

## Run locally

```bash
pip install -r requirements.txt
python main.py
```

Open [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Deploy

Push to any Python-compatible host (Render, Railway, Fly.io, etc.) and set the `PORT` environment variable.
